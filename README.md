# wgBlocks
A simple Xoops module for presentation of information in one or more blocks.

# Require:
- XOOPS 2.5.11;
- XOOPS Admin 1.2;
- PHP 7.4 or higher;
- MySQL 5.5; -> Minimum Recommended Using InnoDB Version 5.1 or Later
